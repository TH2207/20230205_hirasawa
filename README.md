## 作成した目的
・Advanceテストの課題(意見の問い合わせ管理のため)

## アプリケーションURL
・問い合わせ内容登録 → http://127.0.0.1:8000/  
・問い合わせ内容検索/削除 → http://127.0.0.1:8000/find/  

## 機能一覧
・問い合わせ内容の登録  
・問い合わせ内容の検索  
・問い合わせ内容の削除
